# Ícones PWA do BROCRAFT

## Ícones Necessários

Gerar os seguintes ícones a partir do logo do BROCRAFT:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## Como Gerar

1. Use uma ferramenta como [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Ou gere manualmente com Figma/Photoshop

## Especificações

- Fundo: #0a0a0a (preto escuro)
- Cor principal: #ea580c (laranja BROCRAFT)
- Safe zone para maskable: 80% do centro
